# BKH-App

## Bayt AlKhyata Mobile Application

A comprehensive platform for sewing and tailoring education, connecting learners with expert instructors and providing access to courses, products, and a vibrant community.

### Features

- **Authentication & User Management**: Secure OTP-based registration and login with social media integration (Facebook, Google)
- **E-Learning Platform**: Comprehensive courses, quizzes, and certifications
- **E-Commerce Integration**: Product catalog, shopping cart, and secure checkout
- **Points & Rewards System**: Loyalty program with referral codes and discounts
- **Community Features**: Blog posts, discussions, and user interactions
- **Mobile-First Design**: Optimized for Egyptian market with Arabic/English support

### Technical Stack

- **Backend**: Odoo ERP v18 with custom modules and REST API controllers
- **Authentication**: JWT token-based authentication with refresh mechanisms
- **Security**: HTTPS encryption, input validation, and rate limiting
- **Data Format**: JSON request/response with consistent error handling
- **Deployment**: Egypt-specific configuration with local SMS gateway integration
